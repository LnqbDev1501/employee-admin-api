# Employee Admin API

A clean and simple **ASP.NET Core Web API (CRUD)** project for managing employees, built with **Entity Framework Core**, **SQL Server**, and **C#**.  
This project demonstrates backend fundamentals suitable for junior/fresher .NET developer positions, including REST API design, DTO usage, and database interactions with EF Core.
